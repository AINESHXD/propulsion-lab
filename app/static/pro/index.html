PropulsionLab · Pro

A higher-fidelity version, in development.

The free console runs reduced-order cycle analysis in your browser. Pro is the planned next step: heavier models that run server-side, for coursework, design studies and small teams. It is still being built, so this page is a preview, not a checkout.

What Pro is being built to add

Component maps. Compressor and turbine maps with surge margin and corrected-speed effects, replacing the constant-ratio reduced-order matching used in the free off-design solver.

Real-gas chemistry everywhere. Cantera-backed variable-cp and equilibrium products through the whole cycle, not just the optional hot-section correction.

Batch & optimisation. Multi-parameter sweeps and design optimisation run as background jobs, with results you can export.

A real API token. Authenticated programmatic access with usage limits — the same endpoints the console uses, built for scripts.
